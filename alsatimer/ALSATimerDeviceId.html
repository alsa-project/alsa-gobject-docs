Description

A ALSATimerDeviceId is a boxed object to represent the identifier of timer device. The identifier mainly consists of the class of timer device. The other members; the numerical ID of card, device, and subdevice are optional according to the class of timer device.

The object wraps 'struct snd_timer_id' in UAPI of Linux sound subsystem.
