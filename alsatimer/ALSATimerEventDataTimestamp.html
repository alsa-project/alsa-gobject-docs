Description

A ALSATimerEventDataTimestamp is a GObject-derived object to represent event of timer with timestamp. The instance of object is passed to handler for 'handle-event' GObject signal in ALSATimerUserInstance when it's available. The source of timestamp is decided by 'timer_tstamp_monotonic' option of 'snd-timer' kernel module. The object inherits properties of ALSATimerEventData.

The object wraps 'struct snd_timer_read' in UAPI of Linux sound subsystem.
