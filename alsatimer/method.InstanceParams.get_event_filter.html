Description [src]

Get the list of ALSATimerEventType to filter events. This parameter is only effective for target instance with ALSATimerEventDataType.TIMESTAMP.

Parameters

entries An array of ALSATimerEventType*
 

The array with elements for entries of ALSATimerEventType.
 The argument will be set by the function.
 The length of the array is specified in the entry_count argument.
 The instance takes ownership of the data, and is responsible for freeing it.
entry_count gsize*
 

The number of elements in the above array.
 The argument will be set by the function.
error GError **
  The return location for a GError*, or NULL.
