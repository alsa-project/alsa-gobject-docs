Description

A ALSATimerEvent is a boxed object to represent event of timer. The instance of object uses single storage for two types of event data; ALSATimerEventDataTick for and ALSATimerEventDataTstamp. Applications can decide to use one of the two by passing one of ALSATimerEventType to the call of alsatimer_user_instance_attach().
