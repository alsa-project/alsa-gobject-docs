Description

A ALSATimerDeviceInfo is a GObject-derived object to represent information of timer device. The call of alsatimer_get_device_info() returns an instance of the object according to the identifier of timer device.

The object wraps 'struct snd_timer_ginfo' in UAPI of Linux sound subsystem.
