Description [src]

Get the version of timer protocol currently used. The version is represented as the array with three elements; major, minor, and micro version in the order. The length of major version is 16 bit, the length of minor and micro version is 8 bit each.

Parameters

proto_ver_triplet An array of guint16*
 

The version of protocol currently used.
 The argument will be set by the function.
 The array must have 3 elements.
 The data is owned by the caller of the function.
error GError **
  The return location for a GError*, or NULL.
