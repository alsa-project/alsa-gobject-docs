Description

A ALSAHwdepDeviceInfo is a GObject-derived object to represent information of ALSA hwdep device. The call of alsahwdep_get_device_info() returns an instance of the object.

The object wraps 'struct snd_hwdep_info' in UAPI of Linux sound subsystem.
