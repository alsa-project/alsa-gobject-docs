Description

A ALSASeqQueueTimer is a GObject-derived abstract object to represent common properties and methods of timer for queue. Applications can use derived object; ALSASeqQueueTimerAlsa.

The object wraps 'struct snd_seq_queue_timer' in UAPI of Linux sound subsystem.
