Description

A ALSASeqPortInfo is a GObject-derived object to represent information of port. The call of alsaseq_get_port_info() returns the instance of object. The call of alsaseq_user_client_create_port() and alsaseq_user_client_update_port() requires the instance of object.

The object wraps 'struct snd_port_info' in UAPI of Linux sound subsystem.
