alsaseq_get_client_id_list (guint **entries,

Get the list of clients as the numerical ID.

The call of function executes open(2), close(2), and ioctl(2) system calls with SNDRV_SEQ_IOCTL_CLIENT_ID, SNDRV_SEQ_IOCTL_SYSTEM_INFO, and SNDRV_SEQ_IOCTL_QUERY_NEXT_CLIENT command for ALSA sequencer character device.

Parameters

entries

The array with elements for numerical ID of client. One of ALSASeqSpecificClientId can be included in result as well as any numerical value.

[array length=entry_count][out]

The number of entries.
