Description [src]

Get the list of subscription for given address and query type.

The call of function executes open(2), close(2), and ioctl(2) system calls with SNDRV_SEQ_IOCTL_QUERY_SUBS command for ALSA sequencer character device.

Parameters

addr ALSASeqAddr
 

A ALSASeqAddr to query.
 The data is owned by the caller of the function.
query_type ALSASeqQuerySubscribeType
 

The type of query, one of ALSASeqQuerySubscribeType.
entries A list of ALSASeqSubscribeData*
 

The array with element for subscription data.
 The argument will be set by the function.
 The called function takes ownership of the data, and is responsible for freeing it.
error GError **
  The return location for a GError*, or NULL.
