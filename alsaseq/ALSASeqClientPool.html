Description

A ALSASeqClientPool is a GObject-derived object to represent information of pool owned by client. The pool consists of a batch of cells to store message contents in kernel space. The call of alsaseq_get_client_pool() returns the instance of object. The call of alsaseq_user_client_set_pool() and alsaseq_user_client_get_pool() require the instance of object.

The object wraps 'struct snd_seq_client_pool' in UAPI of Linux sound subsystem.
