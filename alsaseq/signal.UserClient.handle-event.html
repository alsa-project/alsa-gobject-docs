Description [src]

When event occurs, this signal is emit with the instance of object which points to a batch of events. The instance should not be passed directly to alsaseq_user_client_schedule_event() again because its memory alignment is different for events with blob data.

Default handler:

The default handler is called after the handlers added via g_signal_connect().

Parameters

ev_cntr ALSASeqEventCntr
 

The instance of ALSASeqEventCntr which points to the batch of events.
 The data is owned by the caller of the function.
