Description

A ALSACtlElemInfoInt is a GObject-derived object to represent the information of integer type of element, which contains integer values. The object inherits methods and properties of ALSACtlElemInfo. A call of alsactl_card_get_elem_info() can returns the instance of object, or a call of alsactl_card_add_elems() requires the instance of object as argument.
