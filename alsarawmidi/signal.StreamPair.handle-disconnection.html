Description [src]

When the sound card is not available anymore due to unbinding driver or hot unplugging, this signal is emit. The owner of this object should call g_object_free() as quickly as possible to release ALSA rawmidi character device.

Default handler:

The default handler is called after the handlers added via g_signal_connect().
