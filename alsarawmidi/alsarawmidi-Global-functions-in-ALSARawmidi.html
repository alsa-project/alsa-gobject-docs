void
alsarawmidi_get_subdevice_id_list (guint card,
                                   guint device,
                                   ALSARawmidiStreamDirection direction,
                                   guint **entries,
                                   gsize *entry_count,
                                   GError **error);

Get the list of numerical IDs for subdevices belongs to the numerical ID of card, device, and the direction.

The call of function executes open(2), close(2), and ioctl(2) system call with SNDRV_CTL_IOCTL_RAWMIDI_INFO command for ALSA control character device.

Parameters

card

The numberical value for sound card to query.

  

device

The numerical value of rawmidi device to query.

  

direction

The direction of stream to query, one of ALSARawmidiStreamDirection.

  

entries

The list of card.

[array length=entry_count][out]

entry_count

The number of entries.

  

error

A GError. Error is generated with domain of g_file_error_quark().
