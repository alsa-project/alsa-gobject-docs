Description

A ALSARawmidiSubstreamInfo is a GObject-derived object to represent information of substream attached to the pair of streams. The call of alsarawmidi_stream_pair_get_substream_info() or alsarawmidi_get_substream_info() return the instance of object.

The object wraps 'struct snd_rawmidi_info' in UAPI of Linux sound subsystem.
